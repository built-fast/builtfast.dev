### New Feature
- Added "Copy for AI" button to every endpoint in the [API reference](/api/)
- Click the sparkles icon to copy an AI-friendly representation including:
    - Documentation index reference
    - Endpoint title and description
    - Complete OpenAPI spec for that endpoint
- Paste into any LLM to generate working integration code in your language of choice

### Documentation Index
- Added `/docs/llms.txt` - a plain text index of all documentation and API endpoints for LLM consumption
