### Breaking Changes
- Event names no longer use `vector.` prefix (e.g., `vector.site.created` â†’ `site.created`)
- Dev container events consolidated under `site.*` namespace

### New Features
- Granular lifecycle events with `.started`, `.completed`, `.failed` phases for provisioning, suspension, unsuspension, and termination
- Deployment rollback events (`deployment.rollback.*`)
- WAF events for rate limits, IP blocks, and referrer rules
- Secret management events
- Domain events
- API key events
- Cache purge events
