date: 2026-01-31 13:00:00 -0500
tags: [vector-pro, breaking]
---

### Breaking Change
- `partner_customer_id` field has been renamed to `your_customer_id` across all API endpoints
- Affected endpoints:
    - `POST /sites` - request body parameter
    - `PATCH /sites/{id}` - request body parameter
    - `POST /sites/{id}/clone` - request body parameter
    - All site and environment response objects

### Action Required
- Update API integrations to use `your_customer_id` instead of `partner_customer_id`
- Requests using the old field name will fail validation
