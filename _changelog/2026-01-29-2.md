## builtfast.dev

### New Features
- Command palette for power-user navigation (`Cmd+K` / `Ctrl+K`)
- Vim-style g-prefix keyboard shortcuts for quick navigation
  - `gh` Home, `gd` Docs, `ga` API, `gb` Blog, `go` Open Source, `gt` Team, `gc` Changelog
- Keyboard shortcuts reference panel (`g?`)
- Theme switching from command palette
