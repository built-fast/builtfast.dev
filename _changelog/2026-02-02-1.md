---
title: "Vector Pro: Database Reset Password Route Renamed"
date: 2026-02-02 11:30:00 -0500
tags: [vector-pro, breaking]
---

The database reset password endpoint has been shortened from `/database/reset-password` to `/db/reset-password` for consistency with other database routes.

<!--more-->

### Affected Routes

| Before | After |
|--------|-------|
| `POST /sites/{site}/database/reset-password` | `POST /sites/{site}/db/reset-password` |
| `POST /environments/{env}/database/reset-password` | `POST /environments/{env}/db/reset-password` |

### Action Required
- Update any API calls using the old `/database/reset-password` paths to use `/db/reset-password`
