### New Feature

- New Backups API for managing on-demand and scheduled site backups
  - `GET /sites/{site}/backups` -- list backups with pagination
  - `GET /sites/{site}/backups/{backup}` -- get backup details
  - `POST /sites/{site}/backups` -- create a new backup with optional description
