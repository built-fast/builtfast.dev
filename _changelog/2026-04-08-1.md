New API endpoints for managing CDN and WAF configurations on Vector Pro sites.

<!--more-->

### New Features

**CDN Management**

- `GET /sites/{site}/cdn/caching` -- Get caching settings (smart cache, vary headers, origin shield, request coalescing, stale content)
- `PUT /sites/{site}/cdn/caching` -- Update caching settings
- `GET /sites/{site}/cdn/optimizer` -- Get optimizer settings (image optimization, asset minification, quality)
- `PUT /sites/{site}/cdn/optimizer` -- Update optimizer settings
- `GET /sites/{site}/cdn/edge-rules` -- List edge rules
- `GET /sites/{site}/cdn/edge-rules/{rule}` -- Get a specific edge rule
- `POST /sites/{site}/cdn/edge-rules` -- Create an edge rule
- `PUT /sites/{site}/cdn/edge-rules/{rule}` -- Update an edge rule
- `DELETE /sites/{site}/cdn/edge-rules/{rule}` -- Delete an edge rule

**WAF Management**

- `GET /sites/{site}/waf/settings` -- Get WAF core configuration (execution mode, profiles, rule groups, payload limits)
- `PUT /sites/{site}/waf/settings` -- Update WAF settings
- `GET /sites/{site}/waf/profiles` -- List WAF profiles
- `GET /sites/{site}/waf/ddos-protection` -- Get DDoS protection settings
- `PUT /sites/{site}/waf/ddos-protection` -- Update DDoS protection settings
- `GET /sites/{site}/waf/bot-detection` -- Get bot detection settings
- `PUT /sites/{site}/waf/bot-detection` -- Update bot detection settings

**WAF Access Lists**

- `GET /sites/{site}/waf/access-lists` -- List access lists
- `GET /sites/{site}/waf/access-lists/enums` -- Get access list enums
- `GET /sites/{site}/waf/access-lists/{accessList}` -- Get a custom access list
- `POST /sites/{site}/waf/access-lists` -- Create a custom access list
- `PUT /sites/{site}/waf/access-lists/config` -- Update access list configuration
- `PUT /sites/{site}/waf/access-lists/{accessList}` -- Update a custom access list
- `DELETE /sites/{site}/waf/access-lists/{accessList}` -- Delete a custom access list

See the [Vector Pro API documentation](/api/) for full details.
