### New Features
- Rebuilt developer portal with auto-generated API documentation
- Vector Pro documentation covering architecture, authentication, deployments, security, databases, environments, domains, and performance
- Full API reference with code examples in curl, PHP, and JavaScript
- Documentation automatically updates from API changes via GitHub Actions

### Developer Tools
- [Vector CLI](/oss/vector-cli/) - Rust-based CLI for site management
- [PHP SDK](/docs/vector-pro/tools/php-sdk/) - Fully typed, PSR-18 compliant
- [Node.js SDK](/docs/vector-pro/tools/node-sdk/) - TypeScript-first with async/await
- [OpenAPI Spec](/api/openapi/) - For code generation and tooling
- [Postman Collection](/api/postman/) - Ready-to-use API requests
