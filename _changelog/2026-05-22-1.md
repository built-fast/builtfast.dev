### Developer Tools

Vector CLI `v0.9.0` is out, rewritten from the ground up in Go.

- New `vector completion` command for Bash, Zsh, Fish, and PowerShell (Homebrew installs completions automatically).
- `go install github.com/built-fast/vector-cli/cmd/vector@latest` now supported for installs from source.

Install via Homebrew (`brew install built-fast/devtools/vector`) or grab a binary from [Releases](https://github.com/built-fast/vector-cli/releases). See the [Vector CLI project page](/oss/vector-cli/) for full details.
