---
title: "Vector Pro: Backups & Restores Promoted to Top-Level API Groups"
date: 2026-02-27 12:00:00 -0500
tags: [vector-pro, breaking]
---

Backups and Restores are now top-level API endpoint groups, no longer nested under Sites.

<!--more-->

### Affected Routes

| Before | After |
|--------|-------|
| `GET /sites/{site}/backups` | `GET /backups` |
| `GET /sites/{site}/backups/{backup}` | `GET /backups/{backup}` |
| `POST /sites/{site}/backups` | `POST /backups` |
| `POST /sites/{site}/backups/{backup}/downloads` | `POST /backups/{backup}/downloads` |
| `GET /sites/{site}/backups/{backup}/downloads/{download}` | `GET /backups/{backup}/downloads/{download}` |
| `GET /sites/{site}/restores` | `GET /restores` |
| `GET /sites/{site}/restores/{restore}` | `GET /restores/{restore}` |
| `POST /sites/{site}/restores` | `POST /restores` |

### Action Required
- Update any API calls using the old `/sites/{site}/backups` paths to use `/backups`
- Update any API calls using the old `/sites/{site}/restores` paths to use `/restores`
