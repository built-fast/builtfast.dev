Initial public release of the Vector Pro developer tools -- the PHP SDK, Node.js SDK, and CLI are now available on GitHub.

<!--more-->

### Developer Tools

- **[Vector Pro SDK for PHP](https://github.com/built-fast/vector-pro-sdk-php)** - Official PHP SDK requiring PHP 8.3+ and a PSR-18 HTTP client. Install via Composer with `composer require built-fast/vector-pro-sdk`
- **[Vector Pro SDK for Node.js](https://github.com/built-fast/vector-pro-sdk-node)** - Official Node.js/TypeScript SDK. Install via npm with `npm install @built-fast/vector-pro-sdk`
- **[Vector CLI](https://github.com/built-fast/vector-cli)** - Command-line interface with pre-built binaries for Linux, macOS, and Windows. Install via Homebrew with `brew install built-fast/devtools/vector`

All three projects are MIT licensed and available at [github.com/built-fast](https://github.com/built-fast). See the [API documentation](/api/) for usage details.
