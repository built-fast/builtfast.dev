---
title: "Vector Pro: Account Summary Caching & 7-Day Deployment Metrics"
date: 2026-04-23 12:00:00 -0500
tags: [vector-pro, breaking]
---

Account summary responses are now cached for 5 minutes, and deployment metrics have switched from a 30-day window to a 7-day window.

<!--more-->

### Breaking Changes

**`deployments_30d` and `failed_deployments_30d` renamed**

The deployment metrics on `GET /vector/account/summary` now report activity over the last 7 days instead of 30 days, and the field names have been updated to reflect the new window.

| Before | After |
| --- | --- |
| `deployments_30d` | `deployments_7d` |
| `failed_deployments_30d` | `failed_deployments_7d` |

### API Improvements

- `GET /vector/account/summary` responses are now cached for 5 minutes per account.

### Action Required

- Update clients reading account summary responses to use `deployments_7d` and `failed_deployments_7d`.

See the [Vector Pro API documentation](/api/) for full details.
