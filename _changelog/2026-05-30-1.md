### New Features

- Every API endpoint is now published as a standalone plain-text file at `/docs/llms/api/<slug>.txt`, containing the endpoint's description and full OpenAPI spec. Agents and LLMs can fetch a single endpoint's spec without copy/pasting from the [API reference](/api/).
- Added an endpoint index at `/docs/llms/api/index.txt` listing every endpoint with its method, path, and a link to its `.txt` file for discovery.
- Linked the new index from `/docs/llms.txt`.

### Developer Tools

- The "Copy for AI" button and the published `.txt` files now share a single pre-rendered source, so clipboard output and fetched files stay in sync.
