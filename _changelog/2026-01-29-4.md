## api.builtfast.com

### Breaking Changes
- Operation IDs now include parent group name to prevent naming collisions
- Format changed from `{subgroup}{action}` to `{group}{subgroup}{action}` (camelCase)

| Before | After |
|--------|-------|
| `databasesResetPassword` | `sitesDatabasesResetPassword` |
| `databasesResetPassword` | `environmentsDatabasesResetPassword` |
| `databasesList` | `sitesDatabasesList` |
| `databasesList` | `environmentsDatabasesList` |

### Action Required
- Regenerate SDK clients built from the OpenAPI spec
- Update any hardcoded operation ID references
