What is Postman?

Postman is a popular API development tool that makes it easy to build, test, and document APIs. It provides a visual interface for making HTTP requests, organizing them into collections, and sharing with your team.

Our collection includes every Vector Pro endpoint with pre-configured authentication, request bodies, and environment variables. Just import and start making requests.

New to Postman? Download the app {% include icons/external-link.html size="xs" %} or use the web version {% include icons/external-link.html size="xs" %}.

Import the collection

Download the JSON file and import it into Postman via File → Import or drag and drop.

postman.json → Postman
2

Set your API token

Create an environment and set the api_token variable to your Vector Pro API key.

Environments → Add Variable
3

Send your first request

Open any endpoint in the collection and hit Send. All requests are pre-configured.

Sites → List Sites → Send

Working with Postman

  • {% include icons/check.html size="sm" %} Use the Console — View → Show Postman Console displays raw HTTP traffic. Essential for debugging.
  • {% include icons/check.html size="sm" %} Save examples — After a successful request, save it as an example for future reference.
  • {% include icons/check.html size="sm" %} Collection Runner — Run multiple requests in sequence to test complete workflows.
  • {% include icons/check.html size="sm" %} Pre-request scripts — Variables like {% raw %}{{site_id}}{% endraw %} are populated automatically from your environment.
